====== 1. basic functions ======
Created понедельник 13 Ноябрь 2017

1. Начало файла инициализация скрипта (script — название файла в unity (script.cs))
{{.\pasted_image.png}}

2. При инициализации сцены (перед стартом) запускается функция Awake()
{{.\pasted_image003.png}}

3. следующая будет Start()
{{.\pasted_image004.png}}

4. для выполнения действия в каждом кардре используют функцию Update() зависит от fps
{{.\pasted_image005.png}}

5. для выполения действяи через определенный промежуток времени (по умолчанию 0.02сек) используют FixedUpdate()
{{.\pasted_image006.png}}


Чтобы добавить новые функции прописываем тип возвращаемого значения и название далее список переменных принимаемых, либо пустые скобки далее кишки функции
