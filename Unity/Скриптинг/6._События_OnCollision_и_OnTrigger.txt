====== 6. События OnCollision и OnTrigger ======
Created среда 15 Ноябрь 2017



Создаем объект (кубик) который может двигаться при нажатии кнопок. В свойствах кубика добавляем компонет Add Component -> Physics -> Rigidbody там убираем Use Gravity чтобы объект не проваливался (земли то нет).
{{./pasted_image.png}}


создаем в иерархии стену из кубика (в scale.x ставим больше значение — кубик удлиняется)
К стене привязываем скрипт scale который будет обрабатывая события столкновений выводить информацию в консоль В свойствах стены добавляем компонет Add Component -> Physics -> Rigidbody там убираем Use Gravity чтобы объект не проваливался (земли то нет).

существуют следующие методы — обработчики событий столкновений:
OnCollisionEnter(Collision collision) — срабатывает каждый раз при начале столкновения (предметы соединились)
OnCollisionExit(Collision collision) — срабатывает каждый раз когда заканчивается столкновение (предметы разъединились)
OnCollisionStay(Collision collision) — срабатывает постоянно пока объекты соприкосаются

чтобы объект стал триггером — ему нужно поставить галочку "**Is trigger**"
OnTriggerEnter(Collision collision) — срабатывает каждый раз при начале столкновения (предметы соединились)
OnTriggerExit(Collision collision) — срабатывает каждый раз когда заканчивается столкновение (предметы разъединились)
OnTriggerStay(Collision collision) — срабатывает постоянно пока объекты соприкосаются
