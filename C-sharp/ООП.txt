====== ООП ======
Created четверг 28 Декабрь 2017

организация программы:
	вокруг кода ------ Код воздействует на данные
	вокруг данных  ------  Данные управляют доступом к коду (Тип данных точно определяет операции, которые могут быть выполненны над данными) — **Это объектно-ориентированный подход**


Для поддержания принципов ООП, язык должен обрадать свойствами:
	инкапсуляцией
	полиморфизмом
	наследованием
